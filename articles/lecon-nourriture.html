Guide complet de la nourriture japonaise

🍜 Plats principaux (Shushoku - 主食)

Riz (Gohan - ご飯)

  • Gohan (ご飯) → Riz cuit / Repas en général
  • Onigiri (おにぎり) → Boulettes de riz
  • Chirashizushi (ちらし寿司) → Riz aux fruits de mer
  • Takoyaki (たこ焼き) → Boulettes de poulpe (Osaka)
  • Omurice (オムライス) → Riz frit dans une omelette

Nouilles (Men - 麺)

  • Ramen (ラーメン) → Nouilles dans un bouillon
  • Udon (うどん) → Nouilles épaisses de blé
  • Soba (そば) → Nouilles de sarrasin
  • Somen (そうめん) → Nouilles fines d'été
  • Yakisoba (焼きそば) → Nouilles sautées
  • Yakiudon (焼きうどん) → Udon sautés

Sushi et poissons

  • Sushi (寿司) → Riz vinaigré + poisson
  • Sashimi (刺身) → Poisson cru sans riz
  • Maki (巻き) → Sushi roulé
  • Nigiri (握り) → Sushi pressé à la main
  • Temaki (手巻き) → Cône d'algue
  • Unagi (うなぎ) → Anguille grillée

🥘 Plats cuisinés populaires

Viandes

  • Yakiniku (焼肉) → Barbecue coréen-japonais
  • Yakitori (焼き鳥) → Brochettes de poulet
  • Tonkatsu (とんかつ) → Porc pané frit
  • Karaage (から揚げ) → Poulet frit japonais
  • Teriyaki (照り焼き) → Grillé sauce sucrée
  • Sukiyaki (すき焼き) → Fondue de bœuf sucrée
  • Shabu-shabu (しゃぶしゃぶ) → Fondue japonaise

Plats à base d'œufs

  • Tamagoyaki (卵焼き) → Omelette sucrée
  • Oyakodon (親子丼) → Bol de riz poulet-œuf
  • Katsudon (カツ丼) → Bol de riz porc pané-œuf

Légumes et tofu

  • Tempura (天ぷら) → Beignets de légumes/crevettes
  • Agedashi tofu (揚げ出し豆腐) → Tofu frit
  • Yasai itame (野菜炒め) → Légumes sautés

🍱 Formats de repas

  • Bentou (弁当) → Repas dans une boîte
  • Teishoku (定食) → Menu complet avec riz, soupe, légumes
  • Kaiseki (懐石) → Cuisine haute gastronomie
  • Izakaya (居酒屋) → Bar à tapas japonais
  • Tabehoudai (食べ放題) → Buffet à volonté
  • Nomihoudai (飲み放題) → Boissons à volonté

🥢 Condiments et assaisonnements

  • Shoyu (醤油) → Sauce soja
  • Miso (味噌) → Pâte de soja fermentée
  • Wasabi (わさび) → Raifort japonais
  • Gari (ガリ) → Gingembre mariné
  • Ponzu (ポン酢) → Sauce agrumes-soja
  • Mayonēzu (マヨネーズ) → Mayonnaise japonaise

🍮 Desserts (Dezāto - デザート)

Traditionnels

  • Mochi (餅) → Gâteau de riz gluant
  • Dorayaki (どら焼き) → Pancakes fourrés
  • Taiyaki (たい焼き) → Pâtisserie en forme de poisson
  • Wagashi (和菓子) → Confiseries traditionnelles

Modernes

  • Purin (プリン) → Crème caramel
  • Keeki (ケーキ) → Gâteau occidental
  • Aisu (アイス) → Glace

🍵 Boissons (Nomimono - 飲み物)

Chaudes

  • Ocha (お茶) → Thé vert
  • Matcha (抹茶) → Thé vert en poudre
  • Kōhii (コーヒー) → Café

Froides

  • Mizu (水) → Eau
  • Ramune (ラムネ) → Limonade japonaise
  • Calpis (カルピス) → Boisson lactée

Alcoolisées

  • Sake (酒/日本酒) → Alcool de riz
  • Bīru (ビール) → Bière
  • Umeshu (梅酒) → Liqueur de prune

🗣️ Expressions essentielles au restaurant

"Osusume wa nan desu ka?" (おすすめは何ですか？)

→ "Que recommandez-vous ?"

"Niku wa taberemasen" (肉は食べられません)

→ "Je ne mange pas de viande"

"Karai mono wa sukidesu" (辛いものは好きです)

→ "J'aime les plats épicés"

🍴 Étiquette à table

"Itadakimasu" (いただきます)

→ Se dit AVANT de manger (obligatoire !)

"Gochisōsama deshita" (ごちそうさまでした)

→ Se dit APRÈS avoir mangé (obligatoire !)

"Oishii desu!" (美味しいです!)

→ "C'est délicieux !"

🍽️ Conseil cultural : Au Japon, faire du bruit en mangeant les nouilles (surtout ramen et soba) est non seulement acceptable, mais c'est un signe d'appréciation ! N'hésite pas à "slurper" tes nouilles 🍜
